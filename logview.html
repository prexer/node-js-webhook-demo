AJAX Log File Viewer - ALFV

This is a very simple, yet powerful, log file viewer.

Each time you click the Generate Log Output button in that page, a line is written to the logfile. Note that by using two windows, we are illustrating how the div in this window may be updated asychronously - that's the AJAX part of this log file viewer. In actual use, you would substitute your own real log file for the simulated logfile named ajax-logfile-viewer.log. If you don't generate any new output, it will appear to stop updating. This is a limit of the simulated logfile, not the Ajax Logfile Viewer.

This page (the one you are reading now) contains buttons that fire two functions - startLog() and stopLog(). These functions will read the log file referenced above & update the log div with the results, one line at a time every two seconds without reloading this page.

To see this in action, click Start Viewer above and then click Generate Log Output in the other window. The Log Viewer div will then update asynchronously with the new log file lines that are written. For best affect, click the Generate button several times. To stop the updating, click the Stop Viewer button.

See http://commavee.com/2007/04/13/ajax-logfile-tailer-viewer/ for code samples and further explanation. This is a working example of the AJAX Log File Viewer discussed there.
